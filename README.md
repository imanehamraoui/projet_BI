# BI-project8-rls-health-warehouse
# 🏥 Projet 8 : Sécurité dynamique (Row-Level Security) pour un entrepôt santé

## 👥 Équipe
- Maroua Lhassouani
- Imane Hamraoui

## 🎯 Objectif
Ce projet consiste à concevoir un entrepôt de données pour un hôpital permettant de partager des indicateurs tout en protégeant les données sensibles des patients.

## 🧱 Architecture
- Entrepôt en étoile (consultations, diagnostics)
- Sécurité Row-Level Security (PostgreSQL)
- Vues sécurisées
- Dashboard BI

## 🔐 Sécurité
- RLS par rôle :
  - Médecin
  - Administratif
  - Chercheur
- Masquage dynamique des données sensibles

## 🛠️ Technologies
- PostgreSQL
- Apache Ranger (optionnel)
- Superset / Power BI
- Elasticsearch (audit)

## 📊 Fonctionnalités
- Filtrage des données selon le rôle utilisateur
- Masquage des informations sensibles
- Audit des accès
